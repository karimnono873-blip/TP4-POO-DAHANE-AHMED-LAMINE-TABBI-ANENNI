1. Objectifs de la Mission Architecturale

L'objectif de ce projet est l'application exhaustive des paradigmes de la Programmation Orientée Objet (POO). Contrairement à la programmation procédurale, la POO nous permet de modéliser le système de paie d'une entreprise nationale sous forme d'entités logiques interagissant entre elles.

// ARCHITECTURE DU SYSTÈME DÉPLOYÉ :
> [Interface] ICalculable : Définit le contrat "calculerSalaire()"
> [Classe Abstraite] Employe : Encapsule les attributs universels (Nom, Salaire de base...)
> [Héritage] Administratif extends Employe : Spécialisation avec primes de poste.
> [Héritage] Standard extends Employe : Traitement classique.

Le modèle garantit la sécurité des données (Encapsulation) grâce aux modificateurs private et protected. L'utilisation d'une interface assure le Polymorphisme, permettant au noyau central de traiter n'importe quel type d'employé (Chef de service, Agent, etc.) de manière uniforme lors du calcul du solde net.

2. Implémentation Logicielle : Core Java

Le code ci-dessous représente la hiérarchie des classes développées pour répondre au cahier des charges (Calcul des primes familiales, primes administratives, sécurité sociale à 7% et retenues d'absences à 0.5%/jour). 

package dz.usthb.automatisation.rh;

/**
 * Contrat garantissant que toute entité possède une méthode de calcul de paie.
 */
interface ICalculable {
    double calculerSalaireNet();
}

/**
 * Classe Abstraite définissant le noyau d'un employé.
 */
abstract class Employe implements ICalculable {
    // Encapsulation stricte
    private String nom;
    private String prenom;
    private String dateNaissance;
    
    // Protected pour l'héritage
    protected double salaireBase;
    protected int nbEnfants;
    protected int joursAbsence;

    // Constructeur paramétré
    public Employe(String nom, String prenom, double base, int enfants, int abs) {
        this.nom = nom;
        this.prenom = prenom;
        this.salaireBase = base;
        this.nbEnfants = enfants;
        this.joursAbsence = abs;
    }

    // Méthodes d'instances communes (Polymorphisme interne)
    protected double getPrimeFamiliale() {
        return nbEnfants * 300.0;
    }

    protected double getRetenues() {
        double secSociale = salaireBase * 0.07;
        double penalite = joursAbsence * (salaireBase * 0.005);
        return secSociale + penalite;
    }

    public String getIdentite() {
        return nom.toUpperCase() + " " + prenom;
    }
}

/**
 * Spécialisation : L'employé administratif avec surcharge des primes.
 */
class Administratif extends Employe {
    private String poste;

    // Appel au constructeur de la classe Mère (super)
    public Administratif(String nom, String prm, double b, int enf, int abs, String poste) {
        super(nom, prm, b, enf, abs);
        this.poste = poste;
    }

    // Redéfinition (Override) de la méthode de l'interface
    @Override
    public double calculerSalaireNet() {
        double primePoste = 0;
        
        if (poste.equals("Chef de service")) {
            primePoste = salaireBase * 0.03;
        } else if (poste.equals("Chef de bureau")) {
            primePoste = salaireBase * 0.02;
        }

        // Salaire Brut Réel = Base + Primes - Retenues
        return salaireBase + primePoste + getPrimeFamiliale() - getRetenues();
    }
}
